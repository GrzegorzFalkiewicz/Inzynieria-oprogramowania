# Library Management Spring Boot

Projekt aplikacji webowej napisanej w **Java 17** i **Spring Boot 3.1.4**, służącej do zarządzania książkami i bibliotekami.  
Aplikacja łączy część backendową, warstwę webową oraz obsługę bazy danych. Projekt obejmuje tworzenie i udostępnianie zasobów przez **REST API**, obsługę widoków **Thymeleaf**, walidację danych, zabezpieczenia oparte o role oraz integrację z bazą danych **MySQL**.

## Opis projektu

Celem projektu było stworzenie aplikacji umożliwiającej zarządzanie danymi dotyczącymi książek i bibliotek z wykorzystaniem nowoczesnego stosu technologicznego Spring.  
W ramach projektu zaimplementowano logikę biznesową, warstwę dostępu do danych, kontrolery REST, widoki po stronie serwera oraz mechanizmy bezpieczeństwa i walidacji.

Projekt rozwija umiejętności związane z:
- tworzeniem aplikacji webowych w Javie,
- projektowaniem REST API,
- pracą z relacyjną bazą danych,
- wykorzystaniem Spring Boot i Spring Security,
- walidacją danych wejściowych,
- organizacją struktury projektu w architekturze warstwowej.

## Najważniejsze funkcjonalności

- zarządzanie książkami i bibliotekami,
- udostępnianie danych przez REST API,
- obsługa formatów **JSON** i **XML**,
- walidacja danych wejściowych,
- widoki generowane z użyciem **Thymeleaf**,
- zapis i odczyt danych z bazy **MySQL**,
- wykorzystanie **JPA / Hibernate**,
- autoryzacja i ograniczanie dostępu z użyciem **Spring Security**,
- monitorowanie aplikacji z użyciem **Spring Boot Actuator**,
- testowanie zapytań HTTP przy pomocy plików `.http`.
